# Social Links Quickshare — Chrome Extension

Social Links Quickshare is a lightweight Chrome extension that allows you to quickly view and copy your social, portfolio, and other profile links with a single click. This extension is perfect for developers, designers, and professionals who frequently share their profiles.

## Features

* View all your social and portfolio links in one popup.
* One-click copy link functionality.
* Add, edit, and delete links directly from the popup.
* Custom icons for popular platforms: Instagram, TikTok, LinkedIn, YouTube, GitHub, and Portfolio.
* Responsive and user-friendly UI.
* Data stored using Chrome's `storage.sync` so it syncs across your signed-in devices.


## Installation (Local / Personal Use)

1. Clone or download this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** (top-right).
4. Click **Load unpacked** and select the folder containing the extension.
5. Pin the extension to the toolbar for easy access.

## Usage

* Click the extension icon in the toolbar.
* View your social links.
* Click the copy icon to copy any link.
* Use the **+ Add / Edit Links** button to manage your links.
